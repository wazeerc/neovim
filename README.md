# Neovim config ⌨️

### Install

Neovim — https://neovim.io/

ripgrep  — https://github.com/BurntSushi/ripgrep (Needed for Telescope's live grep)


Put the `init.lua` file and `lua` folder in the respective locations below:

- Windows (PowerShell):
	- `%USERPROFILE%\AppData\Local\nvim\`

- macOS / Linux:
	- `~/.config/nvim/`
